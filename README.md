# 🧠 ThinkBoard – MERN Stack Notes App

ThinkBoard is a modern **MERN stack note-taking application** that allows users to create, view, update, and delete notes through a clean and responsive interface. Each user automatically gets a **unique session-based workspace**, ensuring notes remain private without requiring a login system.

The application demonstrates full-stack development using **MongoDB, Express.js, React, and Node.js**, along with modern UI components and efficient API communication.

---

# ✨ Features

- 📝 Create, update, and delete notes
- 📂 View notes in a clean card-based layout
- ⚡ Fast React frontend using Vite
- 🔒 Session-based user isolation (no login required)
- 📅 Note creation date formatting
- 🚦 API rate limiting for protection
- 🎨 Modern UI using TailwindCSS + DaisyUI
- 📱 Fully responsive design

---

# 🏗 Tech Stack

## Frontend
- React (Vite)
- TailwindCSS
- DaisyUI
- Axios
- React Router
- React Hot Toast
- Lucide Icons

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Upstash Redis (Rate Limiting)

## Deployment
- Render
